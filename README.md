# 🌐 LocalizationOps AI — 国际化产品运营 Demo 平台

> 粘贴一段产品文案，选择目标市场 → AI 自动做一份完整的国际化运营评估报告。
>
> 从文化适配到合规风险，从翻译质量到市场竞争力，输出带评分和优先级的可落地方案。
>
> 展示的不是"我会写代码"，而是 **"我知道国际化产品运营从哪看、看什么、看完怎么落地。"**

## 🎯 这个项目证明什么

| 面试官看到 | 实际想证明 |
|-----------|-----------|
| 你用 AI 搭了一个本地化审计平台 | 你不是只会等需求，你能用工具主动解决问题 |

## 🚀 核心功能：一键本地化审计

输入产品内容 + 选择目标市场 → AI 自动生成完整的国际化运营评估报告。

覆盖维度：
- 🕌 **文化适配** — 宗教、颜色、符号、幽默等文化敏感问题
- 📖 **本地化质量** — 翻译准确性、术语一致性、风格适配
- 🌍 **产品就绪度** — 日期/货币格式、UI 空间、RTL 支持
- ⚖️ **合规与隐私** — GDPR、CCPA、数据跨境风险
- 📈 **市场竞争力** — 本地用户习惯适配、竞品对比

## 🔧 进阶工具（独立模块）

| 模块 | 用途 |
|------|------|
| 🎯 翻译质量评估 | 多维度 AI 翻译质量评分 |
| 📖 术语一致性检查 | 确保多语言术语翻译一致 |
| 🌍 国际化就绪度评分 | 评估产品在海外市场的就绪程度 |
| ✍️ 双语产品文案生成 | 英文文案 + 中文本地化备注 |
| 🕌 文化敏感度检测 | 特定市场的文化风险检测 |
| 💬 用户反馈分析器 | 海外用户评价自动分类与排优 |
| 📊 运营数据看板 | 国际化产品运营指标可视化 |

## 技术栈

- **Streamlit** — 交互式 Web 应用框架
- **DeepSeek API** — AI 分析引擎
- **Plotly** — 数据可视化
- **Pandas** — 数据处理

## 本地运行

```bash
pip install -r requirements.txt
export DEEPSEEK_API_KEY="your-api-key"
streamlit run app.py
```
